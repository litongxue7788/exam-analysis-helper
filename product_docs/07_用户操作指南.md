# 试卷分析助手 (Exam Analysis Assistant) 用户操作指南

**版本**: V1.5.1
**更新日期**: 2026-01-06

---

## 🚀 快速开始 (3步上手)

### 第一步：上传试卷 (Upload)
1.  **打开首页**: 访问应用地址（线上/本地均可）。
    *   **线上**: `http://123.56.45.212/`
    *   **本地**: `http://localhost:3000/`
2.  **添加图片**:
    *   **电脑端**: 将试卷图片拖入上传区域，或点击“添加图片”选择文件。
    *   **手机端**: 点击上传区域，选择“拍照”或从相册选择。
3.  **调整图片**:
    *   点击缩略图右上角的“旋转”按钮，确保文字方向正确。
    *   支持批量上传多张图片。

### 第二步：智能分析 (Analyze)
1.  **确认信息**: 确保左上角的学生姓名、科目设置正确（可在“设置”中修改）。
2.  **填写访问口令（如有）**: 若点击分析提示“访问口令错误或缺失”，请在右上角“设置”中填写访问口令后重试。
2.  **开始分析**: 点击右下角的 **“生成分析报告”** 按钮。
3.  **等待生成**: 系统将自动进行 OCR 识别和 AI 分析（通常需 10-30 秒），完成后自动跳转至报告页。

### 第三步：复盘与训练 (Review & Train)
1.  **查看报告**:
    *   **成绩概览**: 查看总分、班级排名及进退步情况。
    *   **核心错因**: 点击具体的错因卡片，查看“错因诊断”和“最短改法”。
2.  **针对性训练**:
    *   在错因卡片上点击 **“⚡ 练一练”**，AI 将即时生成 3 道同类变式题。
    *   在“提分计划”区域，勾选每日任务。
3.  **验收小测**:
    *   点击底部的 **“✅ 验收”** 按钮，进行本周错题的集中测试。

## 📱 移动端专属功能 (Mobile Only)

*   **单手操作**: 底部设有悬浮操作栏 (Dock)，方便拇指点击“练习”、“验收”和“导出”。
*   **折叠菜单**: 点击左上角的汉堡菜单 (☰)，可快速切换“错题本”或“家长驾驶舱”。
*   **拍照优化**: 上传时自动压缩图片，节省流量并提升上传速度。

## 💻 桌面端专属功能 (Desktop Only)

*   **HUD 指挥舱**: 宽屏下展示三栏布局（左侧导航、中间报告、右侧任务栏）。
*   **拖拽任务**: 可以将错因卡片直接拖拽到右侧的“今日任务”栏中，快速加入计划。
*   **打印预览**: 点击右上角的“导出”或“打印”按钮，可生成 A4 标准排版的 PDF 报告。

## ❓ 常见问题 (FAQ)

**Q: 分析失败怎么办？**
A: 请检查网络连接，或确认“设置”中的 API Key 是否有效。如果图片模糊，请重新拍摄上传。

**Q: 提示“访问口令错误或缺失 (401)”怎么办？**
A: 说明当前服务开启了访问口令校验。请在“设置”中填写正确的访问口令，再重新点击分析。若你是管理员，请在服务器侧检查 `TRIAL_ACCESS_CODES` / `TRIAL_ACCESS_CODE` 环境变量配置。

**Q: 提示“请求体过大 (413)”怎么办？**
A: 请减少图片数量、优先拍清晰的关键页，或把图片发原图后再上传。若你是管理员，请在服务器侧适当增大 Nginx 的 `client_max_body_size`，并确认后端 JSON 限制已满足大图上传。

**Q: 提示“请求过于频繁 (429)”怎么办？**
A: 说明触发了服务端的频控/额度限制。稍后再试，或减少连续提交次数。管理员可在服务器侧调整频控与额度环境变量。

**Q: 如何添加新的学生档案？**
A: 点击首页右上角的“设置”图标，在“学生档案”区域点击“+”号添加。

**Q: 历史记录在哪里？**
A: 点击首页右上角的“切换考试”图标 (🔄)，即可查看过往的分析记录。

## ✅ 一次完整闭环（建议按这个顺序）

1. 上传试卷图片或导入成绩表
2. 生成分析报告（自动进入报告页）
3. 在报告页点击“练一练”生成针对性练习题
4. 完成训练后点击“验收”，通过则记录沉淀；未通过会触发二次训练
5. 家长侧查看“家长驾驶舱”，关注完成率/通过率/趋势/预告四个维度
